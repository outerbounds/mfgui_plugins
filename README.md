# mfgui_plugins
This repository holds useful plugins that can be used with the Metaflow Graphical User Interface (MFGUI)

You can learn more about plugins by reading the [plugin API guide](https://github.com/Netflix/metaflow-ui/blob/master/plugin-api/README.md), [plugin guide](https://github.com/Netflix/metaflow-ui/blob/master/docs/plugin-guide.md), and [plugin system](https://github.com/Netflix/metaflow-ui/blob/master/docs/plugin-system.md).

See the READMEs in each directory for more information on how to use them.
